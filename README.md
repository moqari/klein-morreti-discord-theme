# Klein Moretti - The Fool

Author: Zam

A Lord of Mysteries inspired Discord theme featuring Klein Moretti and Amon. 
The artist of this fanart is mrthersh
https://danbooru.donmai.us/posts?tags=mrthersh

## Installation

1. Download `Klein-Moretti.theme.css`
2. Place it in your Vencord Themes folder
3. Enable the theme in Vencord
